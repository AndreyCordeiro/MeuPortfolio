Meu Localizador

O Meu Localizador é um sistema para a localização e apreensão de bens. O sistema conta com um aplicativo Mobile para que o Localizador possa encontrar determinado veículo e realizar a apreensão do mesmo. O sistema Web possibilita advocacias gerenciarem todo o fluxo do sistema.


Tecnologias utilizadas:

Backend Java 11 com Spring Boot e Microserviços, banco de dados MySQL, aplicativo mobile em Flutter, Frontend em Angular 14.



Pipehub

Pipehub é um integrador de ecommerce's e marketplaces desenvolvido pela NextAge e VIASOFT, que faz o intermédio entre o Construshow (ERP) e a loja virtual do cliente. Sendo assim o cliente centraliza todas as informações em um só lugar podendo fazer vendas online ou na sua loja física sem se preocupar em dar baixa nos produtos do ecommerce ou importar pedidos manualmente.


Tecnologias utilizadas:

Backend Java 11 e Spring Boot, banco de dados MySQL e Oracle DB, mensagerias com o Google PubSub e RabbitMQ, projeto em Node.js para comunicação com o ERP e Dashboard do usuário em Angular 5.



Ecommerce's integrados:

Vtex, Tray, Moovin, Magento, MKX, Climba, Flexy, Area Central, Cws entre outros.
